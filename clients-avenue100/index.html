Avenue100 Media Solutions

Avenue100 operates CourseAdvisor, an innovative lead generation platform for the higher education business.

Code Red Code worked on all levels of the Rails/PostgreSQL stack to add new functionality improving performance and adding features as the site scaled up to several million monthly visits.

Avenue100 was acquired by Digital Media Solutions Group in 2018 and the site (along with some of Code Red Code's contributions) continues to operate today.
