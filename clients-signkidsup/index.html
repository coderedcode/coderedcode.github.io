SignKidsUp

SignKidsUp was a search tool for parents making it easier to enroll their children in extracurricular activites and summer camps, and a platform for camps and programs with limited resources to reach parents and manage enrollment.

Code Red Code stepped in to help as development efforts with an offshore team were stalling and added features for the Ember.js front end and Ruby on Rails back end.
