Muzit

Muzit is a tool for musicians to engage, identify, and monetize their untapped fan base through monitoring of file sharing activity across the world. Code Red Code has been working on developing a new user-facing site to aggregate and display the data collected by our monitoring network, using a combination of Ruby on Rails and Ember.js.
